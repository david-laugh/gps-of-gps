# gps-of-gps
2022년도 2학기 참빛설계학기 프로젝트
[사용자 지정 GPS 좌표와 위치 정보 빅데이터를 활용한 시각화 애플리케이션 구현]

## Feature
- Google Map Click Event : 사용자가 지정한 위치의 좌표를 획득한다.
    - 지정 위치에 마커 추가 기능.
    - Default Position : 서울
